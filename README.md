# Personal Website

A modern, responsive personal portfolio website for a product designer.

## Features

- Clean and minimalist design
- Fully responsive layout
- Smooth animations and transitions
- Interactive project showcase
- Animated statistics counter
- Mobile-friendly navigation

## Getting Started

To view this website locally, simply open `index.html` in your web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (with serve)
npx serve .
```

## Customization

- Replace the avatar placeholder with your actual photo
- Update the content in `index.html` with your information
- Modify colors in `assets/css/style.css` to match your brand
- Add your own projects to the work section
- Update social links in the contact section

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Google Fonts (Inter)
- Font Awesome Icons
